Parallax-scrolling
responsive WordPress site

E.L. Tettensor approached me with a challenge: she wanted an old-fashioned, dark, and ornate WordPress site to promote her new book, and she really liked parallax scrolling effects. Even though it was my first time, I was confident I could do it—and make it responsive to boot.

Art direction

First, I helped E.L. find an illustrator. I did some preliminary searching, then presented her with five great options. I then worked with the chosen illustrator to create an image that would work well with the website I was planning. Once the illustration was nearly ready, I was able to start creating elements that would match—a logotype of sorts, a handful of icons, smoke textures, and various textures and backgrounds.

WordPress theme

The first hurdle was to build everything so it would pull information from WordPress. I built a template that pulled each content page into a single one-page display, then created an independent layout for the blog posts, post archives, and 404 page. I used Stellar.js to implement parallax effects on each panel, and then interspersed each with intermediary panels and overlaid smoke for decoration's sake. Each panel is carefully arranged to have a unique design, while remaining part of a cohesive whole.

Making it responsive

After some experimentation, I decided that the parallax single-page scrolling effect was unnecessarily complex for a responsive implementation, mostly because the fixed-height panels became too restrictive. As a result, I simplified the design quite a great deal for smaller screens, removing the transitions between panels and allowing each panel to scroll. I carefully evaluated every panel from standardized breakpoints, testing extensively to ensure the site works as well as possible on all sorts of devices.

CSS Animations

As a final touch, I added introductory animations to the first panel. CSS animations are carefully timed so that the effect of a light turning on in the room is given, and then each element slowly fades in, with the smoke the final element.

The final result was an immersive, elegant, and utterly usable site that my client was delighted with.

Custom WordPress site for a spa

Cascades Day Spa is one of Halifax’s longest-running and most well-known day spas. I built them a new website that saw a dramatic increase in customer retention as well as bringing in new customers.

Website Revamp

The clients’ old website was a sorely out-of-date eyesore built on a Contribute platform. It made use of tables and font tags and was difficult to use from both a user and an administrative point of view. The site needed an overhaul to present a professional image and to allow for continued growth.

WordPress as CMS

I used WordPress to build the site, making use of special post types to allow for simple updates. I host the site on my WordPress-optimized VPS, making use of caching, security, and backup plugins to ensure bulletproof stability and lighting-fast performance.

Responsive design

The client wanted a site that was simple, clean, and easy to navigate. I used gentle colours and subtle ornamentation to give a serene feeling. I paid careful attention to ensure that the booking process was as simple as possible, and carefully coded everything to be responsive as well.
